# Chapter Overview

Kubernetes is one of the most widely used open source platforms for orchestrating containers. It automates the deployment, scaling, and management of containerized applications. In this chapter, we will explore the basic architecture of a Kubernetes cluster and the roles of its key components. If you want to dive deeper into Kubernetes fundamentals, consider taking the Linux Foundation's free [_Introduction to Kubernetes (LFS158)_](https://training.linuxfoundation.org/training/introduction-to-kubernetes/) course.

Originally developed at Google and open-sourced in 2014, Kubernetes became the first project donated to the newly formed Cloud Native Computing Foundation (CNCF) with the release of version 1.0. Since then, much of the cloud-native ecosystem—including container runtimes, monitoring tools, and application delivery platforms—has grown around Kubernetes.

Because Kubernetes plays a central role in modern cloud platforms, professionals with Kubernetes administration or development skills are in high demand.
