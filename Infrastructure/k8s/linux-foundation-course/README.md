# Linux Foundation — Kubernetes & Cloud Native Courses

Course materials from the Linux Foundation, cleaned and organized for offline reading.

**Source:** [Linux Foundation Training Portal](https://trainingportal.linuxfoundation.org/)

## Courses

| Course | Description |
|--------|-------------|
| [**Introduction to Kubernetes**](introduction-to-kubernetes/README.md) | From containers to Kubernetes: architecture, Minikube, workloads, Services, storage, ConfigMaps/Secrets, Ingress. |
| [**Cloud Native Essentials (LFS250)**](cloud-native-essentials/README.md) | Cloud native architecture, Kubernetes fundamentals, application delivery, and observability. |

## How to use

- Each course has a **README** with a full table of contents and links to every lesson.
- Each **lesson folder** has its own **README** with a suggested reading order.
- Read lessons in order; do the **Knowledge Check** at the end of each module when available.
- For a quick reference, see **[Kubernetes Cheatsheet](KUBERNETES-CHEATSHEET.md)**.
