Navigating the learning experience

  • Who is the course for?

    This course is designed for existing and aspiring developers, administrators, architects and managers who are new to the world of cloud native technologies and container orchestration.

  • What prerequisites should you have?

    The course has no hard requirements, but it is beneficial if you are familiar with the following topics:

    • Basic understanding of Linux systems and simple command line usage– We recommend the free course Introduction to Linux
    • Basic understanding of what the cloud is
    • Basics of a programming language
    • Knowledge of networking concepts like TCP/IP or DNS.
    • We recommend the free course Introduction to Kubernetes as a good start for your Kubernetes journey
  • What the course prepares you for?

    The course, along with real-world experience and study, will provide the skills and knowledge also tested by Kubernetes and Cloud Native Associate (KCNA) exam.
