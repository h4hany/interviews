# KCNA Final Mock Exam

This mock exam is designed to simulate the real KCNA exam experience, covering all the domains with a similar weightage. It consists of multiple-choice questions to test your understanding of the key concepts.

## Instructions

*   Answer all questions to the best of your ability.
*   There is only one correct answer for each question.
*   The passing score for this mock exam is 75%.

---

**1. Which component of the Kubernetes control plane is responsible for storing the cluster state?**

a) kube-apiserver
b) etcd
c) kube-scheduler
d) kube-controller-manager

**2. What is the primary purpose of a Kubernetes Service?**

a) To run a containerized application
b) To provide a stable endpoint for accessing a set of Pods
c) To manage the lifecycle of Pods
d) To store persistent data

**3. Which of the following is NOT a valid Kubernetes workload resource?**

a) Deployment
b) StatefulSet
c) Container
d) Job

**4. How can you expose a Kubernetes Service to the internet?**

a) By using a Network Policy
b) By creating an Ingress resource
c) By using a PersistentVolume
d) By creating a DaemonSet

**5. What is the role of the `kubelet` component in a Kubernetes cluster?**

a) To schedule Pods onto Nodes
b) To manage the container runtime on a Node
c) To store cluster data
d) To expose the Kubernetes API

**6. Which deployment strategy involves gradually replacing old Pods with new ones, ensuring zero downtime?**

a) Recreate
b) Blue/Green
c) Canary
d) Rolling Update

**7. What is the purpose of a PersistentVolumeClaim (PVC) in Kubernetes?**

a) To request storage from a PersistentVolume
b) To define a storage class
c) To create a storage volume
d) To manage the storage lifecycle

**8. Which of the following is a key principle of cloud-native architecture?**

a) Monolithic design
b) Manual scaling
c) Immutable infrastructure
d) Tight coupling of components

**9. What are the three pillars of observability in a cloud-native environment?**

a) Monitoring, Logging, and Alerting
b) Metrics, Logs, and Traces
c) Performance, Availability, and Scalability
d) Security, Compliance, and Governance

**10. Which CNCF project is a service mesh that provides traffic management, security, and observability for microservices?**

a) Prometheus
b) Fluentd
c) Istio
d) Helm

1.  b) etcd
2.  b) To provide a stable endpoint for accessing a set of Pods
3.  c) Container
4.  b) By creating an Ingress resource
5.  b) To manage the container runtime on a Node
6.  d) Rolling Update
7.  a) To request storage from a PersistentVolume
8.  c) Immutable infrastructure
9.  b) Metrics, Logs, and Traces
